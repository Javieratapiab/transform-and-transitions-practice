CSS Transition Fundamentals | Demos

Transition-property: Specifies name or names of CSS Properties which transitions will be applied. Only properties listed as a value of the transition property will be affected by the animation.

Transition-duration(seconds): Specifies the duration of the animation. You can specify a single duration that will apply to all properties during the transition or multiple values to allow each property to transition over a different period of time.

Transition-delay(seconds): Allows to delay the start of the animation. It also takes values in seconds just like transition-duration.

Transition-timing-function: Includes five different options in order to control the speed of the curve.

CSS Transition and 2D Transform | Demos

Definition: CSS Transform allows elements styled in CSS to be transformed in two-dimensional.

Transition-property: You can change the shape of elements, you can resize them by scaling it up and down, and you can also change the position of elements by making them move around the screen.

Introduction to 2D Transform methods

  • Skew(): skews element along X and Y-axis so this method takes two parameters. The first one is used to target the X-axis, so in this example: Skew(20deg, 10deg) that allows to skews the elements by 20 degrees on the X-axis, then the second parameter will allow to skew the same element by 10 degrees on the Y-axis.

    We can use the methods skewX() and skewY() using positive and negative values.
